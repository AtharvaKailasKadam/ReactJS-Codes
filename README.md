# ReactJS-Codes 🚀

This repository contains a collection of **ReactJS code snippets, mini-projects, and component examples** that serve as a practical reference for learning and experimenting with React.

---

## 📚 About the Repository

This repo is a **learning companion and hands-on coding archive** for anyone diving into React.js. Whether you're a beginner or someone looking to brush up on core concepts, the examples here will help reinforce key principles such as:

- JSX and Functional Components
- Props and State
- Event Handling
- Hooks (useState, useEffect, etc.)
- Conditional Rendering
- List Rendering
- Component Styling

## 🛠️ How to Run the Code

1. Clone the repository  
   ```bash
   git clone https://github.com/AtharvaKailasKadam/ReactJS-Codes.git
   ```

2. Navigate to the folder of the specific project  
   ```bash
   cd ReactJS-Codes/01_FirstReactApp
   ```

3. Install dependencies (if package.json is present)  
   ```bash
   npm install
   ```

4. Run the React app  
   ```bash
   npm start
   ```

> Make sure you have **Node.js** and **npm** installed before running the project.
